# ADRules
广告拦截规则

自动拦截或删除`下载弹窗`、`悬浮按钮`等影响用户体验的元素;

## 注意事项
1. 该规则基于油侯脚本:[AnnoyancesInterception](https://github.com/AirBashX/UserScript);
2. 由于广告拦截规则的限制,个别功能无法生效
3. 有条件的话请安装油猴脚本,替代该脚本
